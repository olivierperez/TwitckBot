# Extension - Ngrok

Auto-start ngrok when TwitckBot starts.

```json5
{
    "extension": "Ngrok",
    "enabled": true,
    "data": {
        "path": "<PATH/TO/ngrok.exe>", // 1
        "name": "TwitckBot",
        "port": 9014
    }
}
```

## Explanation

`// 1` [Download the executable file of ngrok](https://ngrok.com/download) and set its location here.
